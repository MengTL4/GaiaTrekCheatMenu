# GaiaTrekCheatMenu

盖娅迷航内置修改器

基于BepInEx6开发，请先安装BepInEx6

按F9开关菜单

功能：

- 无限攻击距离
- 超级攻速
- 10倍资源获取
- 敌人龟速
- 10倍装备卡获取
- 编辑资源/装备/技能卡
- 任意合成最高级
 

引用dll：

- Assembly-CSharp.dll
- UnityEngine.dll
- UnityEngine.CoreModule.dll
- UnityEngine.IMGUIModule.dll
- 0Harmony.dll
- BepInEx.Core.dll
- BepInEx.Unity.dll
